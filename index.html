How do you start a webpage?

First things first: Set up your boilerplate - make the document type declaration, add the html as the root element (all content/structures are within the opening and closing html tabs), and create the head.

<!DOCTYPE html> --always first line, tells the computer what we are making

<html>

<head>

Within the head, set up your link to the stylsheet. This will add the style elements to your page.

<link href="./styles.css" type="text/css" rel="stylesheet">

Give it a title. The title is displayed in the browser title bar, not necessarily the heading that shows up on the screen

<title></title>

Now, add your content. This will be added within the body tags, and will contain all of the elements, pics, tables, audio, video, etc.

<body>

</body>

The final html tag is like the period on the sentence. It should be the last item in your code.

</html>
